# Файлы для итогового задания

В директории `tests` находятся тесты для проверки API, которое должно быть реализовано в веб-сервере.

Директория `web` содержит файлы фронтенда.

Для логина в планировщик переходите по адресу: "http://localhost:7540/login.html" и вводите пароль "admin"

Собрать образ Docker: docker build -t todo-image .
Запустить контейнер Docker: docker-compose up -d
